# Shortcode Usage

This theme supports structured page building with reusable content sections and shortcodes. The examples below are intentionally generic so the boilerplate can be adapted to any business type.

## Basic Structure

A typical page can include:

- A hero heading
- Introductory copy
- Content sections with images and text
- Accordions or FAQs
- A contact or CTA section

## Example Hero Section

```html
<h1>Design Systems for Growing Teams</h1>
<p>Create a scalable visual and content foundation that stays consistent across every page.</p>
```

## Example Content Section

```html
<h2>What This Service Includes</h2>
<p>Use this section to describe your offer, who it is for, and the outcome it creates.</p>
<img src="https://example.com/project-image.jpg" alt="Project preview" />
```

## Example FAQ / Accordion Content

```html
<h2>Frequently Asked Questions</h2>
<h3>How long does a typical project take?</h3>
<p>Project timelines depend on scope, approvals, and content readiness.</p>

<h3>Can this theme be used for different industries?</h3>
<p>Yes. The default copy is client-neutral and intended to be replaced with your own messaging.</p>
```

## Example About Content

```html
<h2>Meet the Team</h2>
<p>Introduce your business, studio, or organization with a short overview and a few trust-building details.</p>
```

## Notes

- Replace all placeholder text, links, and images before launch.
- Use featured images for page heroes where available.
- Contact, testimonial, and CTA sections are meant to be customized for each project.
